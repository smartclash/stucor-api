Stucor API

Imagine stucor website, but most of it as a JSON API for you to use ;)

Available Endpoints

News

GET - https://stucor-api.vercel.app/api/news

List of all news post made by Stucor. It will be an array of objects and each object will contain title, link and the time of the post
